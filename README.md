# UltraPeakPlayer

A Python terminal video player with local playback, YouTube streaming, HD mode, and retro ASCII rendering.

## Features

- 🎥 Local video playback
- 🌐 YouTube streaming
- 📺 Full HD playback mode
- 🖥️ Retro terminal rendering
- 🎵 Audio synchronization
- 📂 Playlist support

## Requirements

- Python 3.10+
- Windows 10/11
- VLC Media Player (64-bit)

## Installation

```bash
git clone https://github.com/yourusername/UltraPeakPlayer.git
cd UltraPeakPlayer

pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Dependencies

- OpenCV
- NumPy
- yt-dlp
- Streamlink
- python-vlc
- pygame
